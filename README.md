# Calculator - Level 0: Simple Calculator

## Task

Проект реализует простой калькулятор согласно [техническому заданию](https://github.com/palina-mrk/calculator).

## How to run the app

### Установка зависимостей

```bash
npm install
```

### Сборка приложения

Для production сборки (оптимизированная версия):

```bash
npm run build
```

Результат сборки будет находиться в папке `dist/`. В этой папке будут созданы два файла:

- `index.html` - HTML файл приложения
- `main.[hash].js` - оптимизированный JavaScript файл (CSS встроен в JS)

### Разработка

Для запуска в режиме разработки с автоматической пересборкой при изменениях:

```bash
npm run dev
```

Для запуска dev-сервера с hot-reload (приложение будет доступно на http://localhost:8080):

```bash
npm start
```

Сервер автоматически откроет браузер и будет перезагружать страницу при изменениях в коде.

### Проверка кода

Проверка кода с помощью ESLint:

```bash
npm run eslint
```

Автоматическое исправление ошибок ESLint:

```bash
npm run eslint:fix
```

Проверка форматирования с помощью Prettier:

```bash
npm run lint
```

Автоматическое форматирование:

```bash
npm run lint:fix
```

## Структура проекта

### Корневая директория

- `package.json` - конфигурация проекта и зависимости
- `webpack.config.js` - конфигурация Webpack для сборки приложения
- `eslint.config.mjs` - конфигурация ESLint
- `babel.config.js` - конфигурация Babel для транспиляции JavaScript
- `index.html` - HTML шаблон приложения

### `src/` - исходные файлы приложения

#### `src/js/` - JavaScript модули

- `main.js` - точка входа приложения, инициализирует все компоненты
- `calculator.js` - класс `Calculator` - основная логика калькулятора, управление состоянием
- `operations.js` - математические операции (сложение, вычитание, умножение, деление, процент)
- `ui.js` - класс `CalculatorUI` - управление пользовательским интерфейсом, обработка кликов по кнопкам
- `theme.js` - класс `ThemeManager` - управление темами и цветами калькулятора
- `keyboard.js` - класс `KeyboardHandler` - обработка событий клавиатуры

#### `src/css/` - стили приложения

- `style.css` - стили для калькулятора и всех его компонентов

### `dist/` - собранное приложение

Эта папка создается после выполнения `npm run build` и содержит оптимизированные файлы для production.

### `.husky/` - Git hooks

- `pre-commit` - hook, который запускается перед коммитом и проверяет код с помощью ESLint и Prettier

## Функциональность

Калькулятор поддерживает следующие операции:

- **Сложение** (`+`)
- **Вычитание** (`-`)
- **Умножение** (`×`)
- **Деление** (`:`)
- **Процент** (`%`)
- **Смена знака** (`+/-`)
- **Сброс** (`AC`)

### Управление с клавиатуры

- Цифры `0-9` - ввод цифр
- `+`, `-`, `*`, `/` - математические операции
- `=` - вычисление результата
- `%` - вычисление процента
- `.` или `,` - десятичная точка
- `Delete` или `Escape` - сброс калькулятора

### Управление темами

Калькулятор поддерживает настройку цветовой схемы через цветовые пикеры в верхней части интерфейса:

- Цвет фона калькулятора
- Цвет текста
- Цвет кнопок операций (справа)
- Цвет кнопок управления (сверху)
- Цвет кнопок с цифрами

## Технические детали

- Приложение написано на чистом JavaScript без использования фреймворков
- Не используется `eval()` для вычислений
- Не используется объект `Math` для математических операций
- Код разбит на модули по функциональности
- Webpack настроен для оптимизации кода (минификация, tree shaking)
- Настроен ESLint для проверки качества кода
- Настроен Prettier для форматирования кода
- Настроен pre-commit hook для предотвращения коммитов с ошибками
